dotnet
~~~~~~~

Usage
------
Since COGS is written in c# using the .Net framework, all of the COGS `publishers <../../command-line/index.html>`_ 
require that `dotnet <https://www.microsoft.com/net/download>`_ is installed.

Download
---------
* To download the latest version of dotnet go `here <https://www.microsoft.com/net/download>`_ and follow installation instructions.
