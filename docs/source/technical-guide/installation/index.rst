Installing COGS
----------------
COGS can be installed for free at `cogsdata.org <http://cogsdata.org/>`_. 

Dependencies
~~~~~~~~~~~~~
All COGS `publishers <../command-line/index.html>`_ require that `dotnet <dotnet/index.html>`_ is installed. 
However, some publishers have additional dependencies which must also be installed:

.. toctree::

    dotnet
    graphviz
    sphinx
